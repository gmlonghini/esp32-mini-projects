# ESP32 Mini Projects

This repository contains small projects and examples related to ESP32, embedded systems, and electronics.

Its main purpose is to study, experiment with, and demonstrate embedded development concepts in a simple and practical way. The projects may cover different topics such as communication, graphical interfaces, sensors, connectivity, low-power operation, and other common embedded-system subjects.

This content is intended for educational and experimental purposes. It may contain mistakes, simplifications, inaccuracies, or approaches that are not considered best practices for production environments.

Contributions, suggestions, and corrections are always welcome.
